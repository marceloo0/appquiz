## Starting the app

1. Clone this repo using `git clone https://github.com/marceloo0/appquiz.git`
2. Move to the appropriate directory: `cd appquiz` <br />
3. Run `yarn` to install dependencies <br />
4. Run `yarn android` to start locally <br />
